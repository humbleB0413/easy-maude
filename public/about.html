About Easy MAUDE

A free, browser-based search tool for the FDA MAUDE adverse event database — built for medical device professionals who need fast, flexible access to public MDR data.

What is Easy MAUDE?

Easy MAUDE is a web application that makes the FDA's MAUDE (Manufacturer and User Facility Device Experience) database searchable using a simple, readable query syntax. Instead of navigating complex government interfaces or writing raw Lucene queries, you type natural tag-based queries like brand:dexcom event:Malfunction and get results immediately.

The tool runs entirely in your browser. No account required, no data sent to our servers. All search results and history are stored locally on your device using IndexedDB.

Key Features

Who Is It For?

Easy MAUDE was designed for professionals who regularly work with FDA regulatory data:

  • Regulatory Affairs specialists — conducting post-market surveillance and preparing regulatory submissions
  • Medical device R&D engineers — researching adverse events for competing or similar devices
  • Quality and Safety teams — signal detection and risk analysis for device families
  • Clinical researchers — reviewing real-world device performance data
  • Healthcare attorneys and consultants — reviewing MDR reports for specific products

Data Source

All data is fetched directly from the OpenFDA Device Adverse Event API operated by the U.S. Food & Drug Administration. Easy MAUDE does not modify or store this data on any server — it is retrieved in real time from the FDA's public API and displayed in your browser.

The MAUDE database contains reports submitted since 1992. Reports are submitted by device manufacturers, importers, and device user facilities as required by the FDA's Medical Device Reporting (MDR) regulations (21 CFR Part 803).

Technical Details

Easy MAUDE is a pure client-side React application with no backend infrastructure. It uses the browser's IndexedDB for result caching, encrypted cookies for settings, and the OpenFDA public API (CORS-enabled) for all data access. The source uses the MIT license and is open for inspection.
